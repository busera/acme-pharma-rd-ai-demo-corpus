# Article visual assets

These PNG files are the public-safe visuals used in the Medium article about the ACME Pharma R&D AI audit-planning demo corpus.

They are included as flattened article-support images only. They are not implementation diagrams, screenshots of private systems, production audit workpapers, or evidence of audit assurance.

The images were copied from the article publication package and re-saved without source metadata before being added to this public repository.

## Boundary

Use these visuals only as companion material for the public demo corpus and article. They do not disclose prompts, model routing, local paths, runtime configuration, raw generated output folders, logs, or private audit material.
